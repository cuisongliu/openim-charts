# OpenIM Install

## Sealos

```shell
wget https://github.com/labring/sealos/releases/download/v4.2.2/sealos_4.2.2_linux_amd64.tar.gz
tar -zxvf sealos_4.2.2_linux_amd64.tar.gz sealos
chmod a+x sealos 
mv sealos /usr/bin/
```
